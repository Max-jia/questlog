# QuestLog 攻略寫作規則

> 本文件是攻略內容的硬性規則。每次新增或修改 `content/guides/` 下的文章前必讀。

## 配圖規則

1. **禁止跨文章重複配圖**：每篇文章的配圖必須是該文章專屬的，不得與同遊戲其他文章共用同一張截圖。
   - 背景（2026-07-17 記錄）：Subnautica 2 的 35 篇文章輪流使用同 3 張截圖，被 Google 判定頁面同質化，13 篇「已爬取但未收錄」。配圖重複是量產內容的典型特徵，直接影響收錄。
   - 做法:每篇文章使用不同的截圖；截圖不夠時,寧可少配圖,也不要重複。
2. 圖片檔名和 alt 文字要描述圖片實際內容，不用「screenshot-01」這類無意義編號命名新圖。

## 發文節奏規則

3. **禁止爆發式發文**：不要一次性發佈幾十篇。新站期（2026 年內）保持每週穩定新增數篇，讓 Google 看到「持續更新」而非「批量傾倒」。

## 結構規則

4. 避免所有文章使用完全相同的模板結構（相同開頭格式、相同結尾段落）。固定元素（如 Related Guides）保留，但正文結構應隨主題自然變化。
