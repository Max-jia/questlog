### Hub and spoke model

Build one main base in the Shallows. Everything goes here. Then build 2-3 forward outposts near key resources. Each outpost: 1 room, 1 Moonpool, 1 battery charger, 1 locker.

The outposts save travel time. The main base handles everything else.

---

### Early game layout

```
Room 1: Fabricator + lockers + battery charger
Room 2: Moonpool (when unlocked)
Outside: Solar panels facing the surface
```

Keep it small. You will expand later.

---

### Mid game layout

```
Room 1: Crafting area: Fabricator, Modification Station, Processor
Room 2: Storage: wall lockers labeled by material type
Room 3: Moonpool with Vehicle Fabricator
Room 4: Power room: Bioreactor + Power Storage
Outside: Solar panels + Hydroelectric turbines
```

---

### Late game layout

Add a Scanner Room for resource tracking. Add a second Moonpool for a backup Tadpole. Add growbeds for infinite food and rubber. Add a Bio Research Unit for advanced crafting.

---

### Storage tips

Wall lockers hold 20 items. Floor lockers hold 30. Place floor lockers sideways against walls for maximum density. Label everything. After a 2-hour resource run, you will not remember which locker has Titanium.
