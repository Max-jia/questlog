### Quick Answer

Subnautica 2 supports **up to 4-player co-op** with full crossplay between PC (Steam) and Xbox Series X|S. You can convert any single-player save to multiplayer at any time. No split-screen, each player needs their own copy and device.

---

### How to Host a Co-Op Game

1. From the main menu, select **Host Multiplayer**.
2. Choose a save file (or start a new world).
3. Pick **Survival** (normal hunger/oxygen) or **Creative** (unlimited resources).
4. Once in-game, open the pause menu → **Friends List**.
5. Invite friends via platform invites, or generate an **8-digit Friend Code** for cross-platform players.

### How to Join a Friend's Game

1. Select **Join Friends** from the main menu.
2. You'll see active games from your friends list.
3. Or enter a friend's 8-digit Friend Code.
4. The host must be online for you to access their save file.

### Convert Solo Save to Co-Op

You can switch any single-player save to multiplayer (and back) at any time, select the save and toggle the multiplayer option. No progress is lost.

### Shared vs Personal, What the Team Shares

| Shared (Team-Wide) | Personal (Per Player) |
|---|---|
| All base storage lockers | Character inventory (what you carry) |
| Scanned blueprints & Data Entries | DNA Adaptations (Angel Combs) |
| Base structures (anyone can build/delete) | Personal survival stats |
| Crafting recipes unlocked by anyone | Oxygen tank and equipment upgrades |
| Scanner Station resource tracking | |

### 9 Co-Op Tips That Actually Work

### 1. Split Roles Before You Start
Assign jobs early: one player scouts new biomes, one builds and maintains the base, one gathers resources. A fourth handles food/water production. This prevents four people doing the same thing inefficiently.

### 2. Use the Tadpole to Travel Together
The Tadpole submarine fits multiple players. Equip the **Haul Chassis** for extra seats and storage, it turns the Tadpole into a proper co-op vehicle.

### 3. Lure and Scan Predators as a Team
One player distracts a Leviathan while another scans it. Blueprints are shared team-wide, so only one person needs to complete the scan. This makes scanning dangerous creatures much safer than solo.

### 4. Drop Items to Transfer Between Players
Character inventories are personal. To give resources to a teammate, drop them on the ground (or in a base locker). There's no direct trade menu.

### 5. Beacon Everything, There's No Map
Subnautica 2 has no world map. Every player should carry beacons and mark: base locations, resource clusters, dangerous zones, and points of interest. These are visible to the whole team.

### 6. One Scanner Station Per Lobby
The Scanner Station tracks resource nodes, but only one resource can be tracked at a time for the entire lobby. Coordinate with your team on what to scan for.

### 7. Watch Base Permissions
Currently, any player can deconstruct or empty any storage locker. There are no permission controls yet (planned for a future update). Trust your teammates.

### 8. New Players Must Catch Up
Joining an advanced world doesn't give you all the host's unlocks. Each player still needs to complete their own DNA Adaptations (Angel Combs). Leave early-game resources for new joiners.

### 9. Don't Rush the Depths
Even with 4 players, deep biomes are lethal without proper gear. Prioritize oxygen upgrades before venturing below 200m. A team wipe at 300m with no backup save is devastating.
