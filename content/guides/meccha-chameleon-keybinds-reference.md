## Movement

![Movement](/images/meccha-chameleon/screenshot-10.jpg)


| Key | Action |
|-----|--------|
| **W A S D** | Walk (no sprint function exists) |
| **C** | Crouch |
| **Space** (hold) | Climb walls |
| **E** | Move up while on wall |
| **Q** | Move down while on wall |
| **Shift** | Detach from wall |

---

## Camera

| Key | Action |
|-----|--------|
| **Mouse** | Look / Aim |
| **Middle Mouse Button** | Rotate camera around character |
| **Alt + Right-click + move mouse** | Zoom in while painting |
| **4** | Free-roam camera (detached) |
| **5** | Auto-track & follow Seekers |

---

## Painting & Camouflage (Hider)

| Key | Action |
|-----|--------|
| **F** | Open / close paint menu |
| **R** | Open pose wheel |
| **Space** (in paint mode) | Eyedropper, sample color from surface |
| **Right-click + move mouse** | Adjust brush size |
| **V** | Toggle character shadow on/off |
| **3** | X-ray rendering toggle (paint mode) |

---

## Clones

| Key | Action |
|-----|--------|
| **Q** (outside wall mode) | Create decoy clone at current position |
| **X** | Remove all active clones |

Clones have a 30-second cooldown. Max 2 at a time. If a hunter shoots your clone, you are eliminated.

---

## Seeker (Hunter)

| Key | Action |
|-----|--------|
| **Left Mouse Button** | Tag / eliminate hiders |
| **2** | Toggle drawing view |
| **3** | Toggle nameplates on/off |

## Pro Tips

- **Lower mouse sensitivity** for precise eyedropper sampling and smoother seeker scanning.
- Rebind F, R, and Space to easily reachable keys near WASD if your hand size makes the defaults awkward.
- Keybinds may reset after game patches, always check Settings → Controls after an update.
- Push-to-talk is strongly recommended over open mic, accidental breathing reveals your spot.
- There is no sprint/run key. Movement speed is constant for all players.
